# Project Radiant

AI-assisted clinical workflow software that helps physicians detect, qualify, and report findings—starting with radiology.

## Overview

Project Radiant is a human-in-the-loop platform focused on workflow optimization, triage, and report drafting assistance. It is designed to EU MDR standards from day one, with a go-to-market strategy anchored in French teleradiology.
